You must create app that will create for you any sequence of any length.

Requirements:
1) Create menu where user can choose action. For example:
    Welcome to sequence application!
    -----------------------------------------------
    Choose action:
    1. Enter random range
    2. Enter array length
    3. Create array
    4. Print Array
    5. Exit
2) Use "switch" for menu
3) Logic of each action in menu must be written in a separate method
4) Everything must be in endless loop
